Crisp cover images for Libertrade Journal Analytics
================================================

The case study displays cover.png up to ~1152px wide (max-w-6xl). On retina screens
that is effectively ~2300 physical pixels — so a ~1000px-wide PNG will look fuzzy.

Recommended files (same folder):
  cover.png      — at least 1400px wide (ideal: 1920–2400px)
  cover@2x.png   — optional: 2× version for retina (e.g. 3840px wide)

How to capture (best quality):
  1. Open https://libertrade-journal-analystics-demo-eight.vercel.app/
  2. Maximize Chrome on a 1920px+ monitor (or DevTools → device toolbar → Responsive → 1400px width)
  3. Click "Sign in to Demo", set Range to "All Time" if you want full data
  4. Chrome DevTools (F12) → ⋮ → Run command → "Capture full size screenshot"
     OR Windows Snipping Tool / ShareX on the browser window only
  5. Save as PNG (not JPEG). Replace cover.png in this folder.
  6. Hard refresh the portfolio site (Ctrl+Shift+R).

Do NOT use images pasted from chat — they are heavily compressed (~50KB).

Target file size: roughly 300KB–2MB for a full-dashboard PNG at 1920px width.
